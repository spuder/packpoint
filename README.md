# PackPoint

The shipping station software for Tindie / Easy Post





## Usage

Create a `.env` file with your settings. You can use the provided examples. 

`cp .env.sample .env`


Then start the server
`rackup`

localhost:9292

## Development

```
bundle install
```

## Production
```
APP_ENV=production rackup
```
